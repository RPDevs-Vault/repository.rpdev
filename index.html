RPDevs Repository

Welcome to the RPDevs Kodi repository source.

Downloads

  • 📁 repository.rpdevs-1.0.0.zip

How to install:

  1. Open Kodi and go to Settings (gear icon).
  2. Select File manager > Add source.
  3. Select <None> and enter the URL pointing to this repository.
  4. Name the source RPDevs and click OK.
  5. Go back to the main menu and select Add-ons.
  6. Click the box icon (Package Installer) at the top left.
  7. Select Install from zip file.
  8. Click RPDevs and select the repository.rpdevs-1.0.0.zip file.
  9. Wait for the Add-on installed notification.
  10. Select Install from repository > RPDevs Repository to explore addons.
